01분석 배경 — 두 분석은 결이 다른가?

본 보고서의 두 분석은 표면적으로 서로 다른 명제를 다룬다.

메인 분석 (편리한 고립)

독립변수

편의 인프라 (카페·편의점·배달·외식)

명제

편의 인프라가 풍부할수록 5060남이 더 고립된다 (r = +0.570, p < 10⁻³⁷)

메커니즘

환경 → 행동: 인프라가 "혼자서도 살 수 있게 만들어" 사회적 접촉의 강제력 소멸

인과 방향

인프라가 사람을 바꾼다 (수동적)

보강 분석 (도움회피)

독립변수

복지 인프라 (복지관·1인가구지원·50플러스)

명제

복지 시설이 있어도 5060남은 낙인 회피로 이용하지 않는다 (① vs ② 차이 p > 0.8)

메커니즘

행동 → 환경: 사람이 시설을 능동적으로 거부 → 시설이 무력화됨

인과 방향

사람이 인프라를 거부한다 (능동적)

표면적으로는 결이 다른 두 분석으로 보인다. 그러나 두 분석이 같은 종속변수(5060남의 사회적 접촉 행동)를 같은 데이터셋에서 분석하며, 둘 다 5060남의 행동 패턴이 핵심 매개 변수라는 점에서 심층 구조는 같다. 본 보고서는 이 통합 가능성을 정량 검증한다.

02융합 가설 — "회피 생태계" 모델

두 분석을 통합하면 다음 명제가 도출된다.

📌 회피 생태계 모델 (Avoidance Ecosystem Model)

5060 남성은 식별 가능한 사회적 접촉을 회피하고 익명적·자족적 접촉을 선호한다. 이 회피 성향은 두 가지 환경 변수와 동시에 상호작용한다:

  • 편의 인프라(카페·편의점·배달) = 익명적·자족적 접촉 → 선호의 도피처로 작동
  • 복지 인프라(복지관·1인가구지원센터) = 식별 가능한 접촉 → 회피의 대상으로 작동

따라서 두 인프라는 같은 회피 행동의 공급측(도피처)과 수요측(거부 대상)이다. 편의 인프라가 풍부할수록 회피가 깊어지고, 복지 인프라가 풍부해져도 보호 효과는 발생하지 않는다.

[5060 남성의 도움회피·낙인회피 성향] (퇴직·역할 상실·자기은폐·남성성 규범) │ ┌───────────────┴───────────────┐ ↓ ↓ (선호: 익명적 접촉) (회피: 식별 가능한 접촉) ↓ ↓ 편의 인프라 의존 ↑ 복지 인프라 미이용 ↓ ↓ ┌───────────┼───────────┐ │ │ │ │ │ 사회적 접촉의 배달·혼밥 이동·외출 보호 효과 강제력 소멸 자족 행동 감소 발생 안 함 ↓ ↓ ↓ ↓ └───────────┴───────────┴───────────────────┘ ↓ [외출-커뮤니케이션 부재] (보이지 않는 고립) ↓ [고독사 위험 누적]

이 모델의 검증 가능한 예측은 두 가지다.

  1. 매개 효과의 비대칭성: 편의 → 자족 행동 → 관계 회피의 경로는 통계적으로 유의하나, 복지 → 자족 행동 → 관계 회피 경로는 유의하지 않거나 약하다. (분석 1)
  2. 동시 발현의 사분면 집중: 편의 인프라 ↑ 사분면(①·②)에서 두 회피 신호(자족 행동·관계 회피)가 같은 행정동에서 동시에 발현된다. (분석 2)

03변수 정의 — 회피의 두 행동 차원

5060남의 회피 성향을 직접 관측할 수는 없으나, 행정동 단위 행동 지표의 공통 패턴으로 추정할 수 있다. 본 분석은 회피의 두 행동 차원을 다음과 같이 정의한다.

차원정의구성 변수해석
관계 회피 직접적 사회 접촉의 부재 외출-커뮤 모두 적은 비율 (z-score) 회피의 결과적 표현
자족 행동 대면 접촉 회피의 행동 결과 (배달식재료 z) + (-이동횟수 z), 평균 회피의 행동적 표현 (매개 변수)

두 차원의 행정동 단위 상관은 r = +0.609 (p < 10⁻⁴³)로 매우 강하다. 둘은 별개 행동 패턴이지만 공통 잠재 변수(회피 성향)를 공유함을 시사한다.

04분석 1: 매개 모형 (Mediation Analysis)

회피 생태계 모델의 핵심 예측을 검증한다. 편의 → 자족 행동(매개) → 관계 회피의 경로가 통계적으로 유의하고, 복지 → 자족 행동(매개) → 관계 회피의 경로는 유의하지 않다면 두 인프라의 비대칭적 역할이 입증된다.

A분석 설계

  • X (독립변수): 편의 인프라(log) 또는 복지 인프라(log)
  • M (매개변수): 자족 행동 점수
  • Y (종속변수): 관계 회피 점수 (외출-커뮤 부재 z-score)
  • 통제변수: 독거비율, 두 X 변수 상호 통제
  • 방법: Baron-Kenny 3-step + Sobel test + 부트스트랩 95% CI (5,000회)
  • N = 424 행정동
그림 1. 매개 모형의 두 경로. 편의 → 자족행동 → 관계회피는 매개효과가 통계적으로 유의(a·b = +0.194, 매개비율 35%, 부트스트랩 95% CI 0 포함하지 않음). 복지 → 자족행동 → 관계회피는 매개효과가 유의하지 않음(a·b = +0.027, CI에 0 포함). 두 인프라의 효과가 명확히 비대칭이다.

융합 가설 검증 결과

편의 인프라는 자족 행동(매개)을 통해 관계 회피로 강하게 이어진다 (a·b = +0.194, p < .001). 같은 매개 모형에서 복지 인프라는 자족 행동에 영향을 미치지 못하고 매개 효과도 발생하지 않는다 (a·b = +0.027, n.s.). 동일한 매개변수(자족 행동)가 한쪽으로만 작동한다는 사실은 회피 성향이라는 잠재 변수가 두 인프라에 비대칭적으로 반응함을 보여준다. 5060남은 편의를 적극적으로 선택하지만 복지를 적극적으로 거부한다.

05분석 2: 사분면 동시 발현 검증

회피 생태계 모델의 두 번째 예측: 편의↑ 사분면(①·②)에서 두 회피 신호가 같은 행정동에서 동시에 발현된다. 이는 회피 성향이 환경(인프라 분포)에 누적된 흔적임을 시사한다.

두 분석은 결이 다른 게 아니라 같은 빙산(5060남의 회피 성향)의 수면 위(편의 의존)와 아래(복지 미이용)다. 매개 모형은 이 빙산의 작동 메커니즘을, 사분면 분석은 그 환경적 누적 흔적을 보여준다.

정량적으로 정리하면:

  • 편의 인프라는 자족 행동(매개) → 관계 회피로 35% 매개됨 (정량 검증)
  • 복지 인프라는 같은 매개 경로로 작동하지 않음 (정량 검증)
  • 편의↑ 그룹의 50% 이상이 두 회피 신호 동시 발현 (정량 검증)
  • 회피 합 점수가 사분면에 따라 +1.29 ~ +1.55 z 단위 차이 (정량 검증)

네 가지 결과가 모두 한 방향을 가리킨다 — 회피 성향이 두 분석을 통합하는 잠재 변수임이 다중 증거로 지지됨.

07정책 함의 — "카페 같은 복지관" 단일 처방

두 분석을 따로 두면 정책 처방이 분리된다:

  • 편의 분석 단독: "편의 풍부 행정동에 사회 프로그램을 추가하라"
  • 복지 분석 단독: "복지 시설의 진입 장벽을 해체하라"

융합 분석은 이 둘을 단일 처방으로 통합한다.

통합 정책 메시지

5060 남성의 회피 성향이라는 행동 장벽을 해체하라. 편의 인프라(카페·도서관)를 "익명적 사회적 처방 거점"으로 재설계하면 두 마리 토끼를 잡는다:

  • 편의가 만든 자족 환경에 사회적 접촉을 다시 주입한다 (메인 분석 처방)
  • 복지 시설을 외형적으로 카페·도서관에 위장시켜 낙인을 우회한다 (보강 분석 처방)

즉 "카페 같은 복지관" 또는 "복지관 같은 카페"라는 하이브리드 공간이 두 분석의 공통 답이다.

08한계와 향후 과제

분석 한계

  • 인과 추론 불가: 횡단면 데이터로 매개 효과는 측정되지만 인과 방향은 단정 불가. 회피 성향 → 환경 선택(self-selection)의 가능성도 있고, 환경 → 회피 행동의 가능성도 있다. 두 방향이 동시에 작동할 가능성이 가장 크나, 우리 데이터로는 분리할 수 없다.
  • 회피 성향의 직접 측정 부재: 본 분석은 회피 성향을 행동 지표(자족 행동·관계 회피)로 간접 추정했다. Barriers to Help-Seeking Scale 같은 심리 척도를 한국 5060남에게 직접 적용한 데이터가 있다면 잠재 변수를 더 정확히 모델링할 수 있다.
  • 매개 변수의 한계: 자족 행동(배달·이동)이 회피 성향의 모든 면을 포착하지 못한다. 통화·문자·온라인 소통 등 추가 행동 지표가 매개에 포함되면 매개 비율이 더 커질 가능성.
  • 집단 수준 분석: 행정동 집계 데이터로 개별 5060 남성의 회피 행동을 직접 식별할 수 없다.

향후 검증 과제

  • 구조방정식 모형(SEM): 회피 성향을 잠재 변수로 설정하고 다지표 측정 모형으로 정식화. 본 분석의 매개 모형을 SEM으로 확장하면 잠재 변수의 신뢰도(Cronbach's α)와 모형 적합도(CFI, RMSEA)를 직접 평가 가능.
  • 조절 매개 효과 (Moderated Mediation): 회피 성향 강도에 따라 (1) 편의→고립 효과 강화, (2) 복지의 보호 효과 약화가 동시 발생하는지 검증.
  • 실제 시설 이용률 데이터 연계: 종합사회복지관·50플러스센터의 연령·성별 이용자 통계가 있다면 도움회피를 직접 측정 가능.
  • 시계열 분석: 36개월 패널을 활용해 시점 간 변화(편의 인프라 신설 → 회피 행동 변화)를 분석하면 인과 방향 추론 가능.
  • "카페 같은 복지관" 시범 사업 효과 측정: 정책 처방의 실증 검증.
