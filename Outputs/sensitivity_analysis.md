# 민감도 분석 결과

## 개요

가중치를 변경했을 때 인덱스 순위가 얼마나 안정적인지 Spearman 순위상관으로 검증.
rho > 0.95이면 "매우 안정", 0.90~0.95이면 "안정", < 0.90이면 "민감".

## Dependency Index (행정동 419개)

| 시나리오 | Spearman rho | Top10 일치 |
|---------|-------------|-----------|
| 기본 (0.30/0.25/0.15/0.15/0.15) | 1.0000 | 10/10 |
| 균등 (0.20/0.20/0.20/0.20/0.20) | 0.9849 | 9/10 |
| 인프라 강화 (0.45/0.25/0.10/0.10/0.10) | 0.9813 | 8/10 |
| 인프라 약화 (0.15/0.25/0.20/0.20/0.20) | 0.9778 | 9/10 |
| 고립 강화 (0.20/0.40/0.15/0.15/0.10) | 0.9878 | 8/10 |
| 배달+이동 강화 (0.20/0.20/0.25/0.25/0.10) | 0.9806 | 9/10 |

**결론**: 가중치 변경에 매우 안정적 — 결론 유지됨

## Avoidance Index (자치구 25개)

| 시나리오 | Spearman rho | Top5 일치 |
|---------|-------------|----------|
| 기본 (0.30/0.30/0.25/0.15) | 1.0000 | 5/5 |
| 균등 (0.25/0.25/0.25/0.25) | 0.9400 | 4/5 |
| 부정갭 강화 (0.35/0.35/0.15/0.15) | 0.9723 | 4/5 |
| 부정갭 약화 (0.20/0.20/0.35/0.25) | 0.9031 | 3/5 |
| 복지불신 강화 (0.20/0.20/0.40/0.20) | 0.8885 | 3/5 |
| 도움부재 단독 (0.50/0.20/0.15/0.15) | 0.9600 | 4/5 |
| D 제외 3변수 (0.35/0.35/0.30) | 0.9677 | 4/5 |

**결론**: 대체로 유지되지만, 복지불신 비중을 극단적으로 높이면(0.40) rho=0.889로 민감해짐. 자치구 수(N=25)가 적어 Dependency보다 가중치에 더 민감함. D 제외 3변수 버전(0.9677)은 D의 PQ47 계열 중복이 결론에 미치는 영향을 확인함.

## 해석

- **Dependency Index**: 모든 시나리오에서 rho > 0.97 → 가중치 변경에 **매우 안정적**. 행정동 수(N=419)가 충분히 커서 순위가 거의 변하지 않음.
- **Avoidance Index**: 대부분 시나리오에서 rho > 0.90이지만, 복지불신 강화(0.40) 시 rho=0.889, Top5 3/5로 떨어짐. **대체로 유지되나 가중치에 더 민감**하며, 이는 자치구 수(N=25)가 적은 데 따른 자연스러운 변동.
- **D 제외 3변수**: D(네트워크축소)가 A(도움부재)와 동일 PQ47 계열에서 파생되는 중복을 제거한 버전. rho=0.9677로 D 포함 여부가 결론에 영향 없음 확인됨.
- 전체적으로 가중치를 합리적 범위 내에서 변경해도 **Q1 최위험 지역 명단과 전체 순위 패턴은 유지**됨. 다만 Avoidance의 개별 자치구 순위는 가중치 설정에 따라 변동 가능하므로 해석 시 유의 필요.
